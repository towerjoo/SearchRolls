Search Rolls
================

A selection-search extension for Google Chrome, which supports(by default):

1. Baidu
2. Google Hongkong
3. Google Taiwan
4. Bing
5. Baidu Baike
6. Baidu Zhidao
7. Wikipedia
8. Douban Book
9. Douban Movie
10. Zhi Hu

## Features

1. Support 10 SEs 
2. You can make changes to the default SEs as you want in option page
